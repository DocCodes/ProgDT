# ProgDT for Atom
This package provides the ability to insert the date a/o time in the following formats.
 * `2018-07-22`
 * `14:30:32.183+05:30`
 * `2018-07-22T14:30:32.183+05:30`

# Install
## Using `apm`:

```
apm install progdt
```
## Using Atom
Search for `progdt` in Atom Settings (`Settings > Install > Search`).
